# Youtube Clone in ReactJS

## Run the app

```
git clone https://github.com/raviraushanweb/yt-clone.git
```

```
npm install --legacy-peer-deps
```

```
npm start
```

## Visit the website: [YT Clone](https://vermillion-semolina-a08336.netlify.app/)
